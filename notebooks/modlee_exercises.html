Introduction

In this exercise, you will implement the modlee package to: - Document an image segmentation experiment with a pretrained model from torchvision. - Receive and train a recommended model.

It may be helpful to keep the API documentation open.

For best performance, ensure that the runtime is set to use a GPU (Runtime > Change runtime type > T4 GPU).

Installation

First, we need to install modlee and its related packages. Make sure that you have an account and API key from the dashboard. Replace "replace-with-your-api-key" with your API key. Run the following three cells; they will execute successively. This process may take a few minutes, so you can review the examples while waiting.

NOTE: if you receive a message to “restart the runtime in order to use newly installed versions,” you can safely “cancel.”

import os
os.environ['MODLEE_API_KEY'] = "replace-with-your-api-key"
